Just Fucking Use Tmux

You're running a long job over SSH. Your WiFi drops for two seconds. Connection closed. Your process? Dead. Your output? Gone.

This is preventable.

What tmux does

Tmux runs on the server, not in your terminal. Close your laptop, SSH drops, terminal crashes — tmux keeps running. Reconnect later with tmux attach and everything is exactly where you left it.

The commands

# Start tmux
tmux

# Detach (leave it running)
Ctrl+b d

# Reattach
tmux attach

# Split screen
Ctrl+b %    # vertical
Ctrl+b "    # horizontal

That's it. Learn the rest when you need it.

The alternatives suck

Screen — Released 1987. Cryptic. Dead community.

Multiple terminals — No persistence. Laptop closes, everything dies.

iTerm2 splits — Mac only. Can't detach. Useless over SSH.

Stop losing work. Install it.

sudo apt install tmux && tmux
